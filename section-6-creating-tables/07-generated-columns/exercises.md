# Exercise: Generated Columns

## Prerequisites
- Basic table columns

---

## Exercise 1: Computing on the Fly

### Context
Store a computed value that updates automatically when dependent columns change.

### Tasks

#### Task 1.1: Full Name
Create table `people` with `first_name`, `last_name`.
Add a generated column `full_name` that concatenates them.

#### Task 1.2: Rectangle Area
Create table `rectangles` with `width`, `height`.
Add a generated column `area`.
